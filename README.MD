MuJoCo 6-Axis Robotic-Arm Surface Scanning Simulation Demo

MuJoCo 六轴机械臂曲面随形扫描仿真系统

基于 MuJoCo 物理引擎和 CasADi 优化的六轴机械臂（Aubo i5）曲面扫描仿真项目。系统包含一个基于 PyQt5 的图形交互界面，支持路径规划可视化、实时仿真监控以及手眼相机视角的离屏渲染。

文件结构与功能说明

1. sim_gui.py (仿真GUI主程序)

功能：仿真系统的 GUI 前端与主控制逻辑。提供参数设置面板（扫描步长、模式、ROI 区域等）和控制面板。可以实时显示样品模型、生成的扫描路径及当前扫描点高亮。驱动MuJoCo的物理步进，并同步更新摄像头画面。

交互注意：暂停时左右键切换采样点时不能点进样品模型窗口。


2. mujoco_sim.py (仿真后端)

功能：封装 MuJoCo 物理仿真内容（暂初版）。后续添加重力补偿&RL训练接口。

可视化：机器人手眼相机和上帝视角的图像，在场景中动态绘制红色（目标点）和绿色（TCP）指示球。


3. casadi_ik.py (逆运动学求解器)

功能：结合 Pinocchio（动力学库）与 CasADi（非线性优化库）优化后的逆运动学（IK）算法库，使用ipopt求解器，支持热启动以加快实时解算速度。 

主要优化内容：最小化末端位置误差（权重最高）、姿态误差、关节平滑度及各关节平均角度约束。


4. curve_utils.py (路径规划算法)

功能：导入处理 3D 网格模型并生成扫描路径，包含螺旋线投影扫描及弓字形扫描算法，设置了调整参数的接口。

优化：最短扫描路径分段重连、法向量计算与过滤。

5. setup_gui.py

功能：单独预览3D模型采样点设置及扫描路径规划效果，GUI前端可调整扫描各项参数且保存.npy点云坐标

6. resize_model_stl.py

功能：方便统一不同来源3D模型的坐标轴方向及数据单位

7. 模型文件 (*.xml, *.stl)

mjcf文件夹/*.xml：MuJoCo所需的文件，包括主场景文件和机械臂模型文件，可自定义了光照、地板、背景、平台设施以及待扫描物体，以及机械臂物理及性能参数。

models文件夹/*.STL/stl/DAE：所有场景所需物品的碰撞模型或视觉模型。
